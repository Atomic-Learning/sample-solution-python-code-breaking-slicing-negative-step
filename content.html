Part 1: Understanding Negative Steps

code_string = "q!2refrdgho2!c73 h#eg4hfet@f gvdd4e kkfgc1dab,r3fcgh fguthofeYe"

Reverse the entire string and then reverse from the 5th character from the end.

print(code_string[::-1]) # Entire string reversed print(code_string[-5::-1]) # Reverse from the 5th character from the end

Every 2nd character forward and backward.

print(code_string[::2]) # Every 2nd character forward print(code_string[::-2]) # Every 2nd character backward

Part 2: Solve the Puzzle

Extract every third character counting backward from the penultimate character to reveal a hidden message.

hidden_message = code_string[-2::-3] # Start at index -2 (penultimate character), go backward with a step of 3 print(hidden_message)
